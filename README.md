# Sudoku++
A simplistic sudoku game written in C++ using Ultralight and TailwindCSS for the UI.

## To Intsall/Setup
```
chmod +X ./besmart.sh
./besmart.sh setup
./besmart.sh run
```

## Info
To edit the css:
```
./besmart.sh watch {given css file}
```
(Tip: Use a browser & remember to `rebuild` before running again)
